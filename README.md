# MuscleApp
## Installation
Create a .env file in the root directory and add the following:
```
FIREBASE_API_KEY=
FIREBASE_AUTH_DOMAIN=
FIREBASE_PROJECT_ID=
FIREBASE_STORAGE_BUCKET=
FIREBASE_MESSAGING_SENDER_ID=
FIREBASE_APP_ID=
FIREBASE_MEASUREMENT_ID=

PORT="3000" 
```

Run the following commands:
```
npm install
npm start
```
